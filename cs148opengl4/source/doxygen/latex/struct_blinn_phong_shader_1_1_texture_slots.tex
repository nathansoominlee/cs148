\subsection{Detailed Description}
Corresponds to the texture unit that we want to bind the texture to. 

We specify the active texture unit using \href{https://www.opengl.org/sdk/docs/man/html/glActiveTexture.xhtml}{\tt gl\+Active\+Texture}. The texture unit has a different role than the texture target (i.\+e. G\+L\+\_\+\+T\+E\+X\+T\+U\+R\+E\+\_\+2\+D). You can read more about it \href{https://www.opengl.org/wiki/Texture}{\tt here}.
