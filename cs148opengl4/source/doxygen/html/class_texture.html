Detailed Description

Stores the assignment framework's representation of a texture and allows reuse of the underlying OpenGL texture.

Note that this class does not handle loading the texture. To see how a texture is loaded, look at TextureLoader::LoadTexture().

Constructs a texture object.

Parameters
rawDataThe raw pointer to the pixel RGBA data.
widthThe width of the texture.
heightThe height of the texture.

Takes in the raw pixel data for a texture and creates an RGBA texture out of it. Note that only RGBA textures are currently supported. Additionally, we assume the data is stored such it describes the texture from bottom to top, one row at a time (aka rows are contiguous in memory, beginning from the bottom row).
